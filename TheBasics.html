API

What is an API?

APIs (Aplication Program Interface) and what they provide, can vary greatly from host to host. Generally, they provide information or functionality that the host has developed or owns. They then allow you to access and use it on your project via their API setup. The Petfinder API gives you access to their expansive database, as well as many options for gathering and using the information. We will look more closely at these options in a little bit.

If you would like to delve deep into the API specifics, here is a 6-part explanation.

RESTful API

The Petfinder API is a RESTful API. The structure of a RESTful API is based on REST architecture style. This type of API uses HTTP requests to interact with the data.
