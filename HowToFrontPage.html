API How-To-Guide

Need help getting started with the Petfinder APi? We can help..

Petfinder provides an expansive online database of rescue pets who need forever homes.The Petfinder API allows you to utilize this database to fit your specific needs. Filled with information for shelters all over North America, as well as profiles of individual animals needing a home, your search can be as broad or specific as you need. You can allow the user to search for nearby shelters, list the animals currently at a specific shelter, or limit the search to one species or breed of animal. This guide will walk you through how to set up your API and what you can do with it.
