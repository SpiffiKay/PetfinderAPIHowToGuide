API How-To-Guide

Need help getting started with the Petfinder API? We can help.

Petfinder provides an expansive online database filled with information for shelters all over North America, as well as profiles of individual animals. The Petfinder API allows you to utilize this database to fit your specific needs. This guide will walk you through how to access the API with JavaScript, and what you can do with it.

Note: This guide assumes you have a basic understanding of JavaScript and HTML
